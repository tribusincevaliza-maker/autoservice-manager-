# Руководство пользователя AutoService Manager

## 1. Запуск программы

1. Установите Python 3.12 или выше с официального сайта https://www.python.org/
2. Установите зависимости (в терминале PyCharm):
3. Запустите приложение (в терминале PyCharm):
4. Откройте браузер и перейдите по адресу:
http://127.0.0.1:5000
## 2. Главное меню

На главной странице есть три раздела:
- **Клиенты** — управление клиентами
- **Автомобили** — управление автомобилями  
- **Заказы** — управление заказами на ремонт

## 3. Работа с клиентами

### Как добавить клиента:
1. Нажмите на ссылку **"Клиенты"**
2. Нажмите кнопку **"Добавить клиента"**
3. Заполните поля:
- ФИО (например: Иванов Иван Иванович)
- Телефон (например: 89123456789)
- Email (например: ivan@mail.ru)
4. Нажмите кнопку **"Сохранить"**

### Как посмотреть список клиентов:
- После добавления клиент появится в таблице
- Таблица содержит колонки: ID, ФИО, Телефон, Email

## 4. Работа с автомобилями

### Как добавить автомобиль:
1. Нажмите на ссылку **"Автомобили"**
2. Нажмите кнопку **"Добавить автомобиль"**
3. Выберите клиента-владельца из выпадающего списка
4. Заполните поля:
- Марка (например: Toyota)
- Модель (например: Camry)
- Госномер (например: А123ВС)
- Год выпуска (например: 2020)
5. Нажмите кнопку **"Сохранить"**

### Как посмотреть список автомобилей:
- В таблице отображаются: ID, Владелец, Марка, Модель, Госномер, Год

## 5. Работа с заказами

### Как создать заказ на ремонт:
1. Нажмите на ссылку **"Заказы"**
2. Нажмите кнопку **"Создать заказ"**
3. Выберите клиента из списка (автомобили клиента подгрузятся автоматически)
4. Выберите автомобиль
5. Выберите мастера (опционально)
6. Напишите описание проблемы (например: "Не заводится двигатель")
7. Укажите стоимость ремонта
8. Нажмите кнопку **"Создать заказ"**

### Как изменить статус заказа:
1. Откройте список заказов
2. В колонке **"Статус"** выберите новое значение:
- **В работе** — ремонт начат
- **Ожидание деталей** — ждём запчасти
- **Завершён** — ремонт готов

### Как посмотреть список заказов:
- Таблица показывает: ID, Клиент, Автомобиль, Проблема, Статус, Стоимость, Дата

## 6. Частые проблемы и решения

| Проблема | Что делать |
|----------|------------|
| Страница не открывается | Убедитесь, что сервер запущен (в терминале есть строка * Running on http://127.0.0.1:5000) |
| Ошибка "ModuleNotFoundError" | Выполните `pip install Flask Flask-SQLAlchemy` |
| Не добавляется клиент | Проверьте, что все поля заполнены |
| Не подгружаются автомобили | Сначала добавьте клиента, потом автомобиль этому клиенту |
