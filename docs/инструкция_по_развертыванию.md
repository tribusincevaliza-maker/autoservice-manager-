# Инструкция по развёртыванию AutoService Manager

## Системные требования
- Операционная система: Windows 10/11, Linux, macOS
- Python версии 3.12 или выше
- Git (опционально, для клонирования репозитория)

## Установка Python

1. Скачайте Python с официального сайта: https://www.python.org/downloads/
2. Запустите установщик
3. **ВАЖНО:** поставьте галочку "Add Python to PATH"
4. Нажмите "Install Now"

## Скачивание проекта

### Способ 1 — через Git (рекомендуется)
```bash
git clone https://github.com/tribusinessevaliza-maker/autoservice-manager-.git
cd autoservice-manager-
```
## Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
pip install Flask Flask-SQLAlchemy
Запуск приложения

В терминале папки проекта выполните:

```bash
python app.py
```

При успешном запуске вы увидите:

```
База данных создана!
 * Running on http://127.0.0.1:5000
```

Открытие в браузере

Перейдите по адресу:

```
http://127.0.0.1:5000
```

Устранение неполадок

Проблема Решение
Ошибка ModuleNotFoundError: No module named 'flask' Выполните pip install Flask Flask-SQLAlchemy
pip не распознаётся Переустановите Python с галочкой "Add to PATH"
Страница не открывается Проверьте, что сервер запущен (есть строка * Running on http://127.0.0.1:5000)

```
